## REMOVED Requirements

### Requirement: Theme is a build-time decision in v1
**Reason**: Runtime theme editing is now supported via the Preferences dialog with `Help > Preferences...` menu item. Users can edit colors and metrics at runtime and persist changes via Save button.

**Migration**: Use the Preferences dialog's Theme tab to edit theme values at runtime. The `Help > Preferences...` menu item replaces the restriction against runtime theme selection.
